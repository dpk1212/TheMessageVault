# The Message Vault - Landing Page

This project contains the HTML and CSS for the landing page of **The Message Vault** service.

## Description

The Message Vault is a service designed to allow users to sign up loved ones (family, friends, colleagues) to receive encouraging, supportive, or motivational messages curated by our team. These messages can be delivered daily, weekly, or 3x a week based on the chosen plan.

This landing page aims to:
- Explain the service and its value proposition.
- Detail how the service works.
- Present pricing plans.
- Build trust through testimonials (aspirational) and clear FAQs.
- Encourage users to sign up for the service.

## How to View

1.  Ensure you have the following files in the same directory:
    *   `index.html`
    *   `style.css`
    *   (Optional but recommended for full experience) Placeholder images replaced with actual images (e.g., `placeholder-hero-bg.jpg`, icons in the `problem-section`, `placeholder-how-it-works.png`).
    *   (Optional) `favicon.ico` and `favicon-apple.png`.
2.  Open the `index.html` file in any modern web browser (e.g., Chrome, Firefox, Safari, Edge).

## Technologies Used
